Como Desemvolvemos

Para a criação deste site, cada integrante ficou responsável por uma parte específica do projeto, como o front-end, o back-end e o design. Todos compartilharam opiniões sobre o projeto, fazendo assim um site mais completo.

Dividimos a criação deste site em etapas, utilizando o Design Thinking como base inicial. Em seguida, realizamos a prototipagem no Figma para desenvolver a estética do site, complementando com o Adobe Color para escolher uma paleta de cores adequada. Utilizamos diversas referências para aprimorar a composição visual e o Canva para a criação do logotipo. Por fim, empregamos o GitHub e o Visual Studio Code para a programação.
